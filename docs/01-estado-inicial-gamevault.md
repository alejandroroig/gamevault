# Estado inicial de GameVault

Este documento describe el estado del proyecto antes de iniciar la profesionalización del flujo de trabajo.

## Stack actual

- Java 25
- Spring Boot 4
- Maven
- PostgreSQL
- MongoDB
- Redis
- Docker Compose

## Funcionalidades existentes

- API Spring Boot.
- Persistencia relacional con PostgreSQL.
- Persistencia documental con MongoDB.
- Caché con Redis.
- Carga inicial de datos.
- Organización por dominios.
